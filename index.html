所需工具 （单击文字直接跳转）

1.Java8 （极其建议64位！！！32位基本带不动Minecraft）

2.樱落怡然最新客户端

若链接失效 可以直接进我们的鹅裙下载哦~（裙号在左边下面）

3.樱落怡然皮肤站账号

第一次启动游戏步骤（安装好上述1.2.软件的前提下）

1.注册皮肤站账号《======点击跳转

.请记住您注册所使用的 电子邮箱 密码 游戏内角色名

.在您填写完毕注册信息并按下注册按钮之后 我们会发送一封确认邮件至您所填写的电子邮箱

请点击邮件中的链接确认是您本人的邮箱，同时我们会激活您的游戏内账号

2.在您的启动器中添加樱落怡然皮肤站认证

.打开您的HMCL启动器

....安装完启动器之后可能会出现桌面上无快捷方式的情况，您需要将安装目录

（默认是C:\Program Files (x86)\樱落怡然客户端\）

下的 樱落怡然启动器.exe文件 右键>发送至>桌面快捷方式 即可手动创建桌面快捷方式

.打开您的皮肤站用户主页做准备

....若您是第一次启动，请在自动弹出的新建账户界面下的登录方式中选择外置登录

然后在您的皮肤站用户主页下方会找到一个标明 将此按钮拖动至启动器 的按钮，将其直接拖动至启动器后 继续在启动器中登录您的皮肤站账号即可

....若您使用您自己的HMCL客户端 极其建议您使用1.15.2的MC版本(此处版本号最后一次编辑2020/08/20)进行游玩

单击您的HMCL客户端左上角的账户头像旁的更多菜单（图像为三条横线）

然后单击右下角的加号 + ，在您的皮肤站用户主页下方会找到一个标明 将此按钮拖动至启动器 的按钮，

将其直接拖动至启动器后 继续在启动器中登录您的皮肤站账号即可

3.设置您的Java虚拟机内存（游戏启动出现异常请设置此步）

.在启动器左侧的游戏一栏中找到你的游戏版本（樱落怡然官方启动器

将其命名为：生存服），单击这一栏右侧的齿轮按钮

找到 最大内存(MB)一栏，以下是中低配的建议

您的设备总运行内存------最大内存设定值

4G（4000MB左右）---------2048 （不建议带任何材质包）

8G（8000MB左右）---------4096

16G（16000MB左右）--4096或6144或8192

and so on...

4.（第三步成功者请跳过）

部分Win64位用户装过Java32位(x86)版本，这将导致您的Java虚拟机内存最大只能为1024（MB）（会导致游戏体验极差），

.（建议）您需要在Win的系统设置或控制面板中卸载原有的Java，然后安装上面提供的Java8（x64），

然后确认“最大内存”上方的Java路径为64位Java的安装路径

.您也可以在32位的基础上安装64位Java（仍保留32位，占储存），然后

更改“最大内存”上方的Java路径为64位Java的安装路径

.

总·游戏内部分插件使用介绍

欢迎你来游玩樱落怡然 ， 服主顾瑶携全体玩家欢迎你的到来 ， 我们争做创建世界上最民主的服务器 ， 致力于创建最温馨的玩家社区 ， 鼓励玩家自主交易 建立社区 ，

基础：

那么首先我们要明白服务器所自带的一些插件

这里直接给大家放出一些常用的指令

  • /warp 这是传送坐标的一个插件，里面包含了许多服务器指定的地点类似主城和商店
  • /bal 和 /baltop 这两个分别是查询你的钱数和你在全服财富榜的排名
  • /money pay 玩家名 数量 这个指令是你转钱给你指定的一名玩家。玩家可以不在线
  • /res tp 领地名 这是在服务器最常用的领地tp功能，绝大部分的玩家地方都是领地，需要通过这种方式tp
  • /plot h 这是让你回到地皮的一个指令，在前期也是十分实用的，因为在前期没钱圈不起地只能住在地皮.
  • /fhb create 这个是创建红包的指令，当你想要娱乐发红包时可以使用这个指令。
  • /ck open 这个是服务器自带的随身背包系统，当你在外面背包满了就可以使用这个给你额外提供一些空间
  • /kit cd 这个指令是当你的服务器菜单（时钟）坏掉的时候给你一个新的菜单，非常的实用非常的奶思，不过10秒才能领一个。
  • /back 这个指令是让你回到服务器给你最后存储的一个地方，用易懂的方式就是你传回到你传过来的地方。
  • ------------------------------------------------------------------------------------------
  • 这些就是服务器里用的较多的指令，还有的需要你们自己探索，菜单里也有介绍。

技能插件

首先是这个mcmmo插件

Mcmmo插件

Mcmmo插件是一个让你的日常的活动提升技能加强自身的一个插件，有许多种不同的技能这里我们以挖矿为例

挖矿的经验的来源是通过挖掘，在挖矿达到50级和一百级会分别解锁两个子技能，其他技能同理，但这只是初级的子技能，要升级子技能啧需要通过等级的积累，可以看到我现在的双倍掉率和碎石机持续时间是非常久的，至于这些技能的作用可以自己到服务器里研究。

（可以看到，演示者等级是1131级）

技能在服务器里的排名

如果你想清楚地看到自己的技能在服务器里的排名可以使用 /mctop 和 /mcrank

/mctop 是查看全服人技能总和的排名，/mcrank 则是你自己单项技能排名.（如图所示）

领地插件

接下来则是领地插件，这是在服务器内最重要的一个插件

这是在服务器内最重要的一个插件，这是保护你在生存世界建筑的一个插件。（圈你的领地）

如何圈地

本服的圈地工具是木锄头，其他任何东西都是没用的哦。

  • 以下图为例假设我们需要圈一块地
  • 左键点击这个方块作为第一个选取点。将他作为领地的左下角
  • 然后按同样的方式找到领地的右上角右键点击它
  • 完成后就会显示绿色的粒子效果，这粒子效果就是你领地的边界
  • 然后输入/res create 领地名字支付费用创建领地。具体费用群内文件有具体的说明

领地的权限

设定领地权限

创建完领地后我们就要设置领地的权限了，在领地里输入/res set 领地名 ，即可进入设置页面，当然如果你在领地当中就可以直接输入/res set

  • 这就是权限的示意图有许多的种类，把鼠标移到上面就可以查看权限，上面有具体的介绍。
  • 但这是领地的公共权限，当你需要给某个人单独的权限时就需要使用/res pset 领地名 玩家名跳出这个玩家的个人权限。你就可以单独为这个玩家设置他的权限
  • 家插件，这个功能在菜单里就自带，不过你也可以通过指令自己设置 /sethome 【名字】就可以设置自己的家，/home 【名字】则是回到自己的家
